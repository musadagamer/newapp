Connect Firebase

Conversa needs a free Firebase project to store accounts and messages across devices. Follow these steps:

  1. Go to console.firebase.google.com
  2. Create a project (any name)
  3. Click Realtime Database → Create database → Start in test mode
  4. Go to Project Settings → Your apps → Add web app
  5. Copy the databaseURL (ends in .firebaseio.com)
